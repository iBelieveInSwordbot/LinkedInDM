Cloud sync settings

Templates are stored in a GitHub Gist so they're shared between Macs and don't disappear. You need two things:

  1. A Gist ID (Matt already created one — ask him)
  2. A Personal Access Token with gist scope (create at github.com/settings/tokens)

Both are stored locally in your browser only. Nothing leaves your Mac except calls to the GitHub API.
